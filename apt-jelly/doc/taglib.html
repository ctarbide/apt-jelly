Introduction

The following is a reference to the APT-Jelly tag library. You may want to read the section on the APT-Jelly tag library in the reference guide to establish a context for using this tag library reference. You may also way to refer to the API Documentation.

  • annotationValue: Outputs an annotation value.
  • file: Redirects output to a new file.
  • forAllConstructors: Evaluates its body for all constructors of the specified type declaration.
  • forAllFields: Evaluates its body for all fields of the specified type declaration.
  • forAllImportedTypes: Evaluates its body for all imported types of a specified type declaration.
  • forAllMethods: Evaluates its body for all methods of the specified type declaration.
  • forAllNestedTypes: Evaluates its body for all nested types of the specified type declaration.
  • forAllPackages: Evaluates its body for all packages in the source base.
  • forAllParameters: Evaluates its body for all parameters of the specified executable declaration.
  • forAllThrownTypes: Evaluates its body for all thrown types of the specified executable declaration.
  • forAllTypes: Evaluates its body for all type declarations in the source base.
  • ifHasAnnotation: Evaluates its body if the specified declaration has a specified annotation.
  • ifHasDeclaration: Evaluates its body if the specified type mirror has a declaration.
  • javaSource: Redirects its output to a java source file.

annotationValue

Outputs an annotation value. If a declaration is specified, it will be used to lookup the annotation value. Otherwise, this tag must be nested within a declaration loop tag and the current declaration will be used. If a default value is specified, it will be output if either (1) the specified annotation doesn't exist, (2) the specified annotation element doesn't exist, or (3) the value of the specified annotation element is its default value. If no default value is specified, the declared default value of the element will be output (if there is one). Otherwise, nothing will be output.

attribute description required? default
declaration The declaration on which to get the annotation value. Yes, unless nested within a declaration loop tag. N/A
annotation The fully-qualified classname of annotation. Yes (none)
element The annotation element (i.e. method) to output. No "value"
default The default value to output if either (1) the specified annotation doesn't exist, (2) the specified annotation element doesn't exist, or (3) the value of the specified annotation element is its default value. No (none)

file

Redirects its output to a new file, relative to the output directory specified to APT.

attribute description required? default
name The simple name of the file (not the full path). Yes N/A
package The package into which this file should be put. No (none)
charset The character set to use for the output. No (none)

forAllConstructors

Evaluates its body for all constructors of the specified type declaration. If a type declaration is specified, its constructors will be used. Otherwise, this tag must be nested within a declaration loop tag for type declarations and the constructors of the current declaration will be used. It is not illegal to specify an interface type declaration, but there will be no constructors.

attribute description required? default
var The context variable to which to assign the current declaration No (none)
indexVar The context variable to which to assign the current loop index. No (none)
declaration The type declaration containing the declarations over which to iterate. Yes, unless nested within a declaration loop tag for types. N/A
annotation If specified, limit the set of declarations over which to loop to be those that are annotated with this annotation. Annotation name must be fully-qualified. No (none)
annotationVar The context variable to which to assign the specified annotation mirror on the current declaration (specified by the "annotation" attribute). No (none)
includeSuperclasses Whether to include the declarations of superclasses in the set of declarations over which to iterate. No false
includeSuperinterfaces Whether to include the declarations of superinterfaces in the set of declarations over which to iterate. No false

forAllImportedTypes

Evaluates its body for all imported types of a specified type declaration. If a type declaration is specified, its imported types will be used. Otherwise, this tag must be nested within a declaration loop tag for type declarations and the imported types of the current declaration will be used. The list of imported types is calculated--it does NOT correspond to the list of import statements in the source code, but it is generated by iterating over the member declarations of the specified type declaration. This list will include the bounds of any formal type parameters. This list will NOT include annotations.

attribute description required? default
var The context variable to which to assign the current declaration No (none)
indexVar The context variable to which to assign the current loop index. No (none)
declaration The type declaration containing the declarations over which to iterate. Yes, unless nested within a declaration loop tag for types. N/A
annotation If specified, limit the set of declarations over which to loop to be those that are annotated with this annotation. Annotation name must be fully-qualified. No (none)
annotationVar The context variable to which to assign the specified annotation mirror on the current declaration (specified by the "annotation" attribute). No (none)
includeClasses Whether to include classes in the list of declarations over which to iterate. No true
includeInterfaces Whether to include interfaces in the list of declarations over which to iterate. No true
